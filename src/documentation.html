Quel module utiliser pour déplacer le fichier index.html vers le dossier dist ?

Aucun :), webpack est un bundler pas un task runner, il est fait pour packeter les fichiers par pour éxécuter des taches. Pour résoudre le problème, il faut soit utiliser un task runner (gulp, grunt,...) soit faire un bon cp dans la commande build du package.json, solution retenue ici pour sa simplicité
